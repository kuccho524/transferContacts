# サンプルデータ

このディレクトリには、公開用に匿名化した最小サンプルを置いています。

- `export_contacts.csv`
  Google Contacts エクスポート CSV の最小例です。
- `registered_contacts.csv`
  Step 2 で生成される登録用 CSV の例です。
- `contacts.csv`
  GAM などで取得した登録済み連絡先一覧の例です。
- `contactgroups.csv`
  GAM などで取得したラベル一覧の例です。

これらは `tests/test_public_samples.py` の入力としても利用します。
