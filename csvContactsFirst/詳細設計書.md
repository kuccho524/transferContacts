# フェーズ1: 基本機能実装

## 1.1 概要
コマンドライン引数でファイルパスを指定し、エクスポートCSVから連絡先CSVとラベルCSVを生成する基本機能を実装する。

**実装状況**: ✅ 完了

## 1.2 モジュール設計

### 1.2.1 main()
**責務**: プログラムのエントリーポイント

**処理フロー**:
1. コマンドライン引数をパース
2. 入力ファイルの存在確認
3. データ読み込み処理を呼び出し
4. CSV生成処理を呼び出し
5. 処理結果を標準出力に表示

**引数**: なし

**戻り値**: なし

**例外処理**:
- argparseによる引数検証エラー：使用方法を表示して終了
- ファイル不存在エラー：エラーメッセージを表示して終了（exit code: 1）

**実装コード**:
```python
def main():
    parser = argparse.ArgumentParser(
        description='CSVデータ突合ツール - エクスポートデータから連絡先CSVとラベルCSVを生成'
    )
    parser.add_argument('input_file', help='入力ファイル（エクスポートデータのCSVファイルパス）')
    parser.add_argument('contact_output', help='連絡先CSV出力ファイル名')
    parser.add_argument('label_output', help='ラベルCSV出力ファイル名')

    args = parser.parse_args()

    # 入力ファイルの存在確認
    if not Path(args.input_file).exists():
        print(f"エラー: 入力ファイルが見つかりません: {args.input_file}", file=sys.stderr)
        sys.exit(1)

    # データ読み込み
    data = read_export_data(args.input_file)

    # CSV生成
    generate_contact_csv(data, args.contact_output)
    generate_label_csv(data, args.label_output)

    print(f"処理完了: {len(data)}件のレコードを処理しました")
```

### 1.2.2 read_export_data()
**責務**: エクスポートCSVファイルを読み込み、辞書のリストとして返す

**引数**:
- `input_file` (str): 入力ファイルパス

**戻り値**:
- `list[dict]`: レコードのリスト

**処理フロー**:
1. UTF-8エンコーディングでファイルを開く
2. csv.DictReaderでCSVを読み込む
3. 全レコードをリストに格納
4. リストを返す

**例外処理**:
- ファイル読み込みエラー：エラーメッセージを表示して終了（exit code: 1）

**実装コード**:
```python
def read_export_data(input_file):
    records = []
    try:
        with open(input_file, 'r', encoding='utf-8') as f:
            reader = csv.DictReader(f)
            for row in reader:
                records.append(row)
    except Exception as e:
        print(f"エラー: ファイルの読み込みに失敗しました: {e}", file=sys.stderr)
        sys.exit(1)

    return records
```

### 1.2.3 generate_contact_csv()
**責務**: 連絡先登録用CSVを生成する

**引数**:
- `data` (list[dict]): レコードのリスト
- `output_file` (str): 出力ファイルパス

**戻り値**: なし

**処理フロー**:
1. 全レコードを走査して、メールアドレスと電話番号の最大件数を調査

6. 成功メッセージを表示

**例外処理**:

**実装のポイント**:
- `open(encoding='utf-8', newline='')` でBOMなしUTF-8を保証
- `csv.writer(lineterminator='\n')` で改行コードLFを保証
- メール・電話番号が存在しない場合は空文字列を出力

### 1.2.4 extract_emails_and_phones()
**責務**: レコードからメールアドレスと電話番号を抽出する

**引数**:
- `row` (dict): CSVレコード

**戻り値**:
- `tuple`: (emails, phones)
  - `emails` (list[dict]): メールアドレスのリスト [{'label': str, 'value': str}, ...]
  - `phones` (list[dict]): 電話番号のリスト [{'label': str, 'value': str}, ...]

**処理フロー**:
1. メールアドレスを抽出
   - "E-mail 1 - Label", "E-mail 1 - Value" から開始
   - "E-mail 2 - Label", "E-mail 2 - Value", ...と順にチェック
   - カラムが存在し、値が空でない場合のみリストに追加
   - **値に " ::: " が含まれる場合は分割し、それぞれを個別のエントリとして追加**
2. 電話番号を抽出
   - "Phone 1 - Label", "Phone 1 - Value" から開始
   - "Phone 2 - Label", "Phone 2 - Value", ...と順にチェック
   - カラムが存在し、値が空でない場合のみリストに追加
   - **値に " ::: " が含まれる場合は分割し、それぞれを個別のエントリとして追加**
3. (emails, phones) のタプルを返す

**実装例**:
- 入力: Phone 1 - Value = "070-0000-0001 ::: 090-0000-0002"
- 出力: phones = [{'label': 'Mobile', 'value': '070-0000-0001'}, {'label': 'Mobile', 'value': '090-0000-0002'}]

**実装コード**:
```python
def extract_emails_and_phones(row):
    """
    レコードからメールアドレスと電話番号を抽出する
    " ::: " で区切られた複数の値がある場合は、それぞれを個別のエントリとして扱う
    """
    emails = []
    phones = []

    # メールアドレスを抽出
    i = 1
    while f'E-mail {i} - Label' in row or f'E-mail {i} - Value' in row:
        label = row.get(f'E-mail {i} - Label', '')
        value = row.get(f'E-mail {i} - Value', '')
        if value:  # 値がある場合のみ追加
            # " ::: " で区切られた複数の値がある場合は分割
            values = [v.strip() for v in value.split(' ::: ') if v.strip()]
            for val in values:
                emails.append({'label': label, 'value': val})
        i += 1

    # 電話番号を抽出
    i = 1
    while f'Phone {i} - Label' in row or f'Phone {i} - Value' in row:
        label = row.get(f'Phone {i} - Label', '')
        value = row.get(f'Phone {i} - Value', '')
        if value:  # 値がある場合のみ追加
            # " ::: " で区切られた複数の値がある場合は分割
            values = [v.strip() for v in value.split(' ::: ') if v.strip()]
            for val in values:
                phones.append({'label': label, 'value': val})
        i += 1

    return emails, phones
```

### 1.2.7 parse_labels()
**責務**: ラベル文字列を分割してリストで返す

**引数**:
- `labels_str` (str): ラベル文字列（例: "Team A ::: Role X ::: * myContacts"）

**戻り値**:
- `list`: ラベルのリスト

**処理フロー**:
1. labels_strが空の場合は [] を返す
2. ":::" で分割（スペースの有無を問わない）
3. 各要素をトリム
4. 空文字列を除外
5. ラベルのリストを返す

**実装例**:
- 入力: "Team A ::: Role X ::: * myContacts" → 出力: ["Team A", "Role X", "* myContacts"]
- 入力: "* myContacts" → 出力: ["* myContacts"]
- 入力: "" → 出力: []

**実装コード**:
```python
def parse_labels(labels_str):
    if not labels_str:
        return []

    parts = labels_str.split(':::')
    labels = [part.strip() for part in parts if part.strip()]

    return labels
```

### 1.2.8 get_label_column_name()
**責務**: ラベル列の列名を取得（PrimaryLabel, SecondaryLabel, ThirdLabel...）

**引数**:
- `index` (int): インデックス（0始まり）

**戻り値**:
- `str`: 列名

**処理フロー**:
1. インデックス0-21は辞書から取得:
   - 0 → 'PrimaryLabel'
   - 1 → 'SecondaryLabel'
   - 2 → 'ThirdLabel'
   - 3 → 'FourthLabel'
   - ...
   - 21 → 'TwentyThirdLabel'
2. 22以降は数値表記:
   - 22 → '23rdLabel'
   - 23 → '24thLabel'
   - ...

**実装コード**:
```python
def get_label_column_name(index):
    ordinals = {
        0: 'PrimaryLabel',
        1: 'SecondaryLabel',
        2: 'ThirdLabel',
        3: 'FourthLabel',
        # ... 省略 ...
        21: 'TwentyThirdLabel',
    }

    if index in ordinals:
        return ordinals[index]
    else:
        num = index + 1
        if num % 10 == 1 and num % 100 != 11:
            return f'{num}stLabel'
        elif num % 10 == 2 and num % 100 != 12:
            return f'{num}ndLabel'
        elif num % 10 == 3 and num % 100 != 13:
            return f'{num}rdLabel'
        else:
            return f'{num}thLabel'
```

### 2.2.1 setup_logging()
**責務**: ログ設定を初期化し、ログファイルと標準出力へのログ記録を設定する

**引数**:
- `log_level` (str): ログレベル（'DEBUG', 'INFO', 'WARNING', 'ERROR'）、デフォルトは 'INFO'

**戻り値**:
- `logging.Logger`: 設定済みのロガーインスタンス

**処理フロー**:
1. logsディレクトリが存在しない場合は作成
2. ログファイル名を生成（convert_YYYYMMDD_HHMMSS.log）
3. ログフォーマットを設定
4. ファイルハンドラを作成（ログファイルへの出力）
5. コンソールハンドラを作成（標準出力への出力）
6. ロガーにハンドラを追加
7. ログレベルを設定
8. ロガーを返す

**実装コード（案）**:
```python
def setup_logging(log_level='INFO'):
    """
    ログ設定を初期化

    Args:
        log_level: ログレベル（'DEBUG', 'INFO', 'WARNING', 'ERROR'）

    Returns:
        logging.Logger: 設定済みのロガー
    """
    import logging
    from datetime import datetime
    from pathlib import Path

    # logsディレクトリを作成
    log_dir = Path('logs')
    log_dir.mkdir(exist_ok=True)

    # ログファイル名を生成
    timestamp = datetime.now().strftime('%Y%m%d_%H%M%S')
    log_file = log_dir / f'convert_{timestamp}.log'

    # ログフォーマット
    log_format = '%(asctime)s - %(levelname)s - %(message)s'
    date_format = '%Y-%m-%d %H:%M:%S'

    # ロガーを取得
    logger = logging.getLogger('csv_converter')
    logger.setLevel(getattr(logging, log_level.upper()))

    # ファイルハンドラ（UTF-8で保存）
    file_handler = logging.FileHandler(log_file, encoding='utf-8')
    file_handler.setLevel(logging.DEBUG)
    file_handler.setFormatter(logging.Formatter(log_format, date_format))

    # コンソールハンドラ
    console_handler = logging.StreamHandler()
    console_handler.setLevel(logging.INFO)
    console_handler.setFormatter(logging.Formatter(log_format, date_format))

    # ハンドラを追加
    logger.addHandler(file_handler)
    logger.addHandler(console_handler)

    logger.info(f'ログファイル: {log_file}')
    logger.info(f'ログレベル: {log_level.upper()}')

    return logger
```

```python
def select_files():
    """
    ファイル選択ダイアログを表示してファイルパスを取得

    Returns:
        tuple: (input_file, contact_output, label_output)
               キャンセルされた場合は None
    """
    import tkinter as tk
    from tkinter import filedialog

    # ルートウィンドウを作成（非表示）
    root = tk.Tk()
    root.withdraw()

    # 入力ファイルを選択
    input_file = filedialog.askopenfilename(
        title="エクスポートCSVファイルを選択",
        filetypes=[("CSV files", "*.csv"), ("All files", "*.*")]
    )

    if not input_file:
        print("キャンセルされました")
        return None

    # 連絡先CSV出力先を選択
    contact_output = filedialog.asksaveasfilename(
        title="連絡先CSVの保存先を選択",
        defaultextension=".csv",
        filetypes=[("CSV files", "*.csv"), ("All files", "*.*")],
        initialfile="連絡先.csv"
    )

    if not contact_output:
        print("キャンセルされました")
        return None

    # ラベルCSV出力先を選択
    label_output = filedialog.asksaveasfilename(
        title="ラベルCSVの保存先を選択",
        defaultextension=".csv",
        filetypes=[("CSV files", "*.csv"), ("All files", "*.*")],
        initialfile="ラベル.csv"
    )

    if not label_output:
        print("キャンセルされました")
        return None

    # ルートウィンドウを破棄
    root.destroy()

    return input_file, contact_output, label_output
```

7. データ読み込み処理を呼び出し（ログに件数記録）
8. CSV生成処理を呼び出し（ログに成功/失敗記録）
9. 処理結果を標準出力とログに記録

**実装コード（案）**:
```python
def main():
    parser = argparse.ArgumentParser(
        description='CSVデータ突合ツール - エクスポートデータから連絡先CSVとラベルCSVを生成'
    )
    parser.add_argument('input_file', nargs='?', help='入力ファイル（エクスポートデータのCSVファイルパス）')
    parser.add_argument('contact_output', nargs='?', help='連絡先CSV出力ファイル名')
    parser.add_argument('label_output', nargs='?', help='ラベルCSV出力ファイル名')
    parser.add_argument('--log-level', default='INFO',
                       choices=['DEBUG', 'INFO', 'WARNING', 'ERROR'],
                       help='ログレベル（デフォルト: INFO）')

    args = parser.parse_args()

    # ログ設定を初期化
    logger = setup_logging(args.log_level)
    logger.info('=' * 60)
    logger.info('CSVデータ突合ツール 処理開始')
    logger.info('=' * 60)

    # コマンドライン引数が指定されているかチェック
    if args.input_file and args.contact_output and args.label_output:
        # コマンドライン引数を使用
        input_file = args.input_file
        contact_output = args.contact_output
        label_output = args.label_output
        logger.info('コマンドライン引数からファイルパスを取得')
    else:
        # ファイル選択ダイアログを使用
        logger.info('ファイル選択ダイアログを表示')
        result = select_files()
        if result is None:
            logger.info('ユーザーによる処理キャンセル')
            print("処理を中止しました")
            sys.exit(0)
        input_file, contact_output, label_output = result
        logger.info('ファイル選択ダイアログからファイルパスを取得')

    # ファイルパスをログに記録
    logger.info(f'入力ファイル: {input_file}')
    logger.info(f'連絡先CSV出力: {contact_output}')
    logger.info(f'ラベルCSV出力: {label_output}')

    # 入力ファイルの存在確認
    if not Path(input_file).exists():
        logger.error(f'入力ファイルが見つかりません: {input_file}')
        print(f"エラー: 入力ファイルが見つかりません: {input_file}", file=sys.stderr)
        sys.exit(1)

    # データ読み込み
    logger.info('データ読み込み開始')
    data = read_export_data(input_file)
    logger.info(f'{len(data)}件のレコードを読み込みました')

    # CSV生成
    logger.info('CSV生成開始')
    try:
        generate_contact_csv(data, contact_output)
        logger.info('連絡先CSVの生成完了')

        generate_label_csv(data, label_output)
        logger.info('ラベルCSVの生成完了')
    except Exception as e:
        logger.error(f'CSV生成中にエラーが発生: {e}', exc_info=True)
        raise

    # 処理完了
    logger.info('=' * 60)
    logger.info(f'処理完了: {len(data)}件のレコードを処理しました')
    logger.info('=' * 60)
    print(f"処理完了: {len(data)}件のレコードを処理しました")
```
